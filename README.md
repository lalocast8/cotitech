# CRM Local

CRM básico hecho en HTML/CSS/JavaScript, sin dependencias externas.

## Funciones incluidas
- Gestión de contactos (alta y eliminación)
- Gestión de empresas (alta y eliminación)
- Gestión de oportunidades (alta, cambio de etapa y eliminación)
- Gestión de tareas (alta, completar/reabrir y eliminación)
- Dashboard con métricas rápidas
- Persistencia con `localStorage`
- Botón para cargar datos de ejemplo

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Comienza a registrar información en cada módulo.
3. Los datos se guardan automáticamente en tu navegador.

## Archivos
- `index.html`: interfaz principal
- `styles.css`: estilos visuales
- `app.js`: lógica del CRM
